Join Features

The Join Features task works with two layers of features or tables. Join Features joins attributes from one feature to another based on spatial, temporal and attribute relationships, or some combination of the three. The tool determines all input features that meet the specified join conditions and joins the second input data set to the first. If you specify a one-to-one join all of the features in data set two that match a feature in data set one have their statistics calculated. In this situation the number of of joined features in data set two are calculated. If a feature in data set one had five matches in data set two, you would see one output feature with a count of five. You may optionally included other statistics on fields in data set two, such as minimum, maximum or mean, that are calculated from the features in data set two. You may also specify a one to many join, where every possible match is maintained in your output, so that if a feature in data set one has 5 matches in data set two, there will be five features in the output, and the raw data is joined to data set one, not summary statistics.

Feature join can be applied to points, lines, polygons and tables without a geometry. You can optionally add a temporal join, as long as both input data sets are time enabled, either as time instants or time intervals.
